Git

  • El comando Git Status se usa para verificar el estado de nuestro repositorio ejecutamos.
  • El comando Git Add acepta otro tipo de parametros que nos seran utiles para trabajar con varios archivos a la vez.
  • El comando Git Commit confirma los cambios del area de stage y los guarda en el repositorio.
  • El comando Git Log se usa para revisar el historial de nuestro repositorio.
